# Arduino Hard Rest Game 🕹️🔌

A hardware-driven game using physical sensors and hardcoded logic.

## ✨ Features
- **Sensor-Controlled:** Real-time input from physical hardware.
- **Hardcoded Logic:** Fast, deterministic calculations (No AI).
- **Serial Bridge:** Seamless communication via `PySerial`.
- **Zero Latency:** Direct hardware-to-game synchronization.

## 🛠️ Tech Stack
- **Python** (Game Logic)
- **Arduino/C++** (Sensor Data)
- **PySerial** (Communication)
